# 🚀 AI Code Debugger

### Intelligent Code Analysis & Debugging using LLMs

🔗 **Live Demo:** https://ai-code-debugger-vb2f.onrender.com

📂 **Repository:** https://github.com/2303A52110/AI-Code-Debugger

---

## 🧠 Overview

**AI Code Debugger** is a web-based intelligent assistant that analyzes source code, identifies issues, and provides structured feedback using advanced Large Language Models (LLMs).

It is designed to simulate how an expert developer reviews code — delivering insights on correctness, efficiency, and best practices.

---

## ✨ Key Features

* 🔍 **Syntax Error Detection** — Identifies syntax-level issues
* 🧠 **Logical Analysis** — Detects flawed logic and edge cases
* ⚠️ **Runtime Risk Prediction** — Highlights potential runtime failures
* 🛠️ **Auto Code Correction** — Suggests improved code
* 📖 **Multi-Level Explanation**

  * Simple (beginner-friendly)
  * Detailed (deep technical insights)
* ⚡ **Optimization Suggestions**
* 📊 **Time Complexity Estimation**
* 🎯 **Code Quality Scoring (out of 10)**
* 💡 **Learning Tips for Improvement**

---

## 🏗️ System Architecture

```text
User Input (Code)
        ↓
Flask Backend (app.py)
        ↓
Prompt Engineering Layer
        ↓
OpenRouter API (LLM)
        ↓
Structured Response Processing
        ↓
Frontend Rendering (HTML/CSS)
```

---

## 🛠️ Tech Stack

| Layer       | Technology           |
| ----------- | -------------------- |
| Backend     | Python, Flask        |
| Frontend    | HTML, CSS            |
| AI Engine   | OpenRouter (LLM API) |
| Deployment  | Render               |
| HTTP Client | Requests             |
